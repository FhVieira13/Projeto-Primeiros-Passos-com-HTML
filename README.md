📘 Projeto — Primeiros Passos com HTML
📌 Sobre o projeto

Este projeto foi desenvolvido como parte de um desafio de aprendizado com o objetivo de praticar os conceitos básicos de HTML. A página apresenta conteúdos introdutórios sobre estrutura HTML, funcionamento de clients e servers, listas e navegação com links.

O foco principal é compreender a organização de uma página web e o uso correto das principais tags HTML.

🎯 Objetivos do projeto

Praticar a estrutura básica de um documento HTML

Utilizar diferentes tags de texto

Criar listas ordenadas e não ordenadas

Trabalhar com links e âncoras internas

Aplicar formatações como negrito, itálico e marcação

Organizar conteúdo de forma semântica

🧱 Estrutura utilizada

O projeto utiliza:

Títulos (h1 até h2)

Parágrafos (p)

Listas (ul e ol)

Links (a)

Destaques (strong, i, u, mark)

Subscrito e sobrescrito (sub, sup)

Citações (blockquote)

Separadores (hr)

🧭 Funcionalidades

✅ Índice com navegação interna
✅ Links que retornam ao topo da página
✅ Links externos abrindo em nova aba
✅ Exemplos de listas ordenadas e não ordenadas
✅ Uso de múltiplas tags HTML

📂 Como executar

Baixe ou copie o arquivo index.html

Clique duas vezes no arquivo

O navegador abrirá a página automaticamente

Não é necessário instalar nada — apenas um navegador web.
